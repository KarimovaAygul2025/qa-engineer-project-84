# QA Engineer Project 84

## Описание

Учебный проект QA-инженера на платформе Hexlet. Содержит документацию по тестированию SPA-приложения интернет-магазина косметики и парфюмерии.

## Структура проекта

| Файл | Назначение |
|------|------------|
| `about.yml` | Профиль автора (имя, должность, навыки) |
| `requirements.yml` | Функциональные, нефункциональные и неявные требования |
| `test-cases.yml` | Тест-кейсы |
| `bugreports.yml` | Баг-репорты |
| `testing-report.yml` | Отчёт о тестировании |
| `regress-report.yml` | Регрессионный отчёт |
| `.github/workflows/hexlet-check.yml` | CI/CD (проверка проекта Hexlet) |

## Требования к системе

### Функциональные
- Поиск товаров по ключевому слову и категории
- Фильтрация товаров
- Просмотр каталога и карточек товаров
- Добавление в корзину и просмотр корзины
- Переключение языка интерфейса (RU/EN)
- Постраничный вывод каталога
- Отображение названия, рейтинга, цен (со скидкой и без)
- Удаление товара из корзины, оформление заказа с переходом на оплату

### Нефункциональные
- Удобство использования и скорость работы
- Безопасность и защита данных
- Мобильная версия
- Совместимость с последними версиями браузеров
- Масштабируемость под нагрузку
- SPA без перезагрузки страницы
- Хранение корзины в сессии браузера

### Неявные
- Нет отрицательных цен и количества
- Совпадение цен в корзине и на странице товара
- Отображение дополнительных сборов до подтверждения заказа
- Возможность очистки корзины
- Корректная индикация наличия товара

## Баг-репорты

1. **Кнопка «Перейти к оплате» не реагирует** — Android 14, Chrome Mobile, Samsung Galaxy S23  
2. **Нет реакции на тап по изображению товара** — iOS 17.4, Safari, iPhone 14  
3. **Некорректная локализация на английский** — macOS, Chrome, MacBook Pro  

## Тест-кейсы

1. **Поиск товаров** — поиск по ключевому слову «массаж»  
2. **Отображение скидки** — проверка расчёта цены со скидкой (крем, 5000₽, 50%)  

## Результаты тестирования

- Поиск товаров — **pass**
- Отображение скидки — **fail**

## Регрессионное тестирование

- Добавление товара с выбором оттенка — pass  
- Применение промокода — pass  
- Работа фильтров по брендам — pass  
- Переход на страницу оплаты — fail  
- Отображение карточки на мобильном — pass  
- Переключение языка в футере — fail
